# Development Context – Aether Web Dev Tools

## Internet Situation
This project was developed entirely **offline** due to:
- Widespread internet restrictions in Iran
- Whitelisting protocols during regional tensions
- Limited access to GitHub, PyPI, and other development resources

## Development Environment
- All code written locally in Visual Studio Code
- Dependencies researched and downloaded when intermittent connectivity allowed
- 100% of the work (28 tabs, ~15,000+ lines of code) completed offline

## Assets Not Uploaded
The following are excluded from this repository due to bandwidth limitations:
- Compiled `.exe` binaries (too large for limited connectivity)
- Screenshots and video demonstrations (will be added in the future)
- Large image assets (icons, logos – kept minimal)

## Verification
Anyone can verify the authenticity of this work by:
1. Cloning the repository
2. Running `pip install -r requirements.txt`
3. Executing `python main.py`

The full application runs as documented. No binaries are required.
